# CV Interactivo

Este proyecto corresponde a un Currículum Vitae Interactivo desarrollado con HTML, CSS, Bootstrap, JavaScript y jQuery.

## Características
- Diseño responsivo con Bootstrap
- Navbar de navegación
- Secciones de perfil, experiencia, formación y habilidades
- Cambio de tema claro/oscuro
- Animaciones al hacer scroll
- Formulario de contacto con validación en tiempo real
- Cambio de color en una sección mediante botón

## Tecnologías usadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
